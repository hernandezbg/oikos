¿Cómo funciona?
  1. Selecciona la caja a la que quieres invitar usuarios
  2. Elige el rol que tendrán (Tesorero o Colaborador)
  3. Define cuántos días será válido el código
  4. Indica cuántas personas pueden usar el código
  5. Comparte el código generado con las personas que quieres invitar
  6. Esas personas usarán el código en la página de registro para unirse a la caja
